# 📚 Prática de Lógica: Variáveis e Interação (Portugol)

Este repositório reúne exercícios iniciais de lógica de programação desenvolvidos em **Portugol**. O foco destas atividades é compreender como declarar variáveis, atribuir valores e realizar a comunicação básica entre o software e o usuário.

---

## 📂 Descrição dos Exercícios

### 1. 👤 Coleta de Dados de Aluno
Este algoritmo simula um sistema de cadastro simples que interage com o usuário em tempo real.
* **Objetivo:** Praticar a leitura de dados via teclado e a exibição de informações formatadas.
* **Lógica:** 1. O sistema solicita a idade (`inteiro`) e o nome (`cadeia`).
    2. Utiliza o comando `leia()` para capturar as entradas.
    3. Exibe os dados concatenados em uma mensagem final utilizando quebras de linha (`\n`).

### 2. 📍 Atribuição de Localidade
Exercício focado na inicialização de variáveis e organização de saída de dados.
* **Objetivo:** Demonstrar como armazenar valores fixos em variáveis e exibi-los.
* **Lógica:** 1. Duas variáveis do tipo `cadeia` (`cidade` e `ondeEstuda`) são inicializadas com o valor "contagem".
    2. O programa imprime os valores na tela informando onde o usuário mora e estuda.

---

## 🛠️ Tecnologias e Conceitos Aplicados

* **Linguagem:** Portugol (Ambiente Portugol Studio).
* **Tipos de Dados:**
    * `inteiro`: Para números inteiros, como a idade.
    * `cadeia`: Para textos e conjuntos de caracteres, como nomes e cidades.
* **Comandos de Entrada e Saída:**
    * `escreva()`: Para exibir mensagens e resultados no console.
    * `leia()`: Para receber informações digitadas pelo usuário.
* **Formatação:** Uso de caracteres de escape como `\n` para organizar o texto em múltiplas linhas.

---

## ⚙️ Como Executar

1.  Certifique-se de ter o **Portugol Studio** instalado.
2.  Crie um novo arquivo no editor.
3.  Copie e cole o código do exercício desejado.
4.  Pressione a tecla `F9` para compilar e rodar o programa.
